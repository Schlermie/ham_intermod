How to make a CSV file

A CSV (Comma-Separated Values) file is a plain text file for storing a table (rows and columns) of data. Each line in a CSV file represents a row of data in your table. Every line of text in the CSV file is separated by the same number of commas to represent columns of text in your table.

For example, say you want to create a CSV file for the following table with 4 rows and 3 columns:

RxFreq
TxFreq
Label
145.27
-
W6ASH
147.485
Staging
144.67
446.5
Crossband Uplink
Since a CSV file is just a plain text file, there are numerous ways to create it. I'll illustrate three of the most common ways.

1. Use a Text Editor

Every computer has a text editor available. Notepad is the default text editor in Windows. MacOS has TextEdit. Linux has vi. Run your favorite text editor and start typing your rows of data with columns separated by commas. For the example above, you should get something like this:
Note the following:
  • Our table has 4 rows so our CSV file has 4 lines.
  • Our table has 3 columns so every row in our CSV file has 2 commas, making 3 columns per row.
  • The first (header) line has no formatting (such as bold text).
  • The columns in our text file aren't necessarily lined up. That's okay.
  • The third line has 2 commas next to each other because the second column in that row is an empty cell.
  • The last line has text with whitespace, "Crossband Uplink". The whitespace does NOT create a new column. Columns are only separated by commas.
Now save your file as a plain text file type. Do NOT save your file as a file type that supports additional formatting (bold, italic, etc.) such as Rich Text Format (RTF) or Word (DOC or DOCX). Either .txt or .csv are common filename extensions to use, but HamIM doesn't care what filename extension you use.

2. Use Microsoft Excel (or LibreOffice Calc)

If you're familiar with using Microsoft Excel, you can use it to create a CSV file. After you type your table into Excel, it will look something like this:
Save your file as a "CSV (Comma delimeted) (*.csv)" file type. Do NOT save it as "CSV UTF-8 (Comma delimited) (*.csv)" or "Excel Workbook (.xlsx)" or any other richly formatted file types.

NOTE: If you don't have access to Microsoft Excel, LibreOffice Calc is an open source alternative to Microsoft Excel that's free to download and use on Windows, macOS, and Linux.
